# 페어봇 사용설명서 (초보자용)

페어봇은 한국투자증권(KIS) OpenAPI로 **KODEX ETF 4종목**을 대상으로
"적응형 변동성 돌파(Adaptive Volatility Breakout)" 전략을 자동 실행하는 봇입니다.

- **전략 요약**: (09:00 시가 기준) 레버/인버스 중 하나만 MA5 위일 때만 거래 승인 → 목표가 돌파 시 진입(09:00~13:00) → 트레일링 스탑 or 15:15 강제 청산
- **안전장치(필수)**: `KIS_LIVE_ENABLED`, `KIS_LIVE_CONFIRM`, `KIS_KILL_SWITCH`, `STOP_TRADING.flag`

> ⚠️ 면책
> - 본 문서는 투자 조언이 아닙니다.
> - 실거래는 반드시 충분한 검증 후 진행하세요.

## 2) .env 설정 (API 키 입력) — 절대 GitHub에 올리면 안 됩니다

### 2.1 .env 만들기

```bash
cd ~/Desktop/kis_adaptive_vb_bot
cp .env.example .env
nano .env
```

### 2.2 반드시 채워야 하는 값

```env
KIS_APP_KEY=...
KIS_APP_SECRET=...
KIS_ACCOUNT_NO=12345678
KIS_ACCOUNT_PRODUCT_CODE=01
```

### 2.3 실전 주문 안전장치(3중 게이트)

실제 주문이 나가려면 아래 조건이 **모두** True여야 합니다.

```env
KIS_LIVE_ENABLED=1
KIS_LIVE_CONFIRM=YES
KIS_KILL_SWITCH=0
```

그리고 프로젝트 루트에 아래 파일이 **없어야** 합니다.

```bash
test -f STOP_TRADING.flag && echo "STOP_TRADING ON" || echo "STOP_TRADING OFF"
```

- 주문을 막고 싶으면:
```bash
cd ~/Desktop/kis_adaptive_vb_bot
touch STOP_TRADING.flag
```
- 다시 허용하려면:
```bash
cd ~/Desktop/kis_adaptive_vb_bot
rm -f STOP_TRADING.flag
```

## 4) 자동 켜고/끄기 (권장 운영)

이 PC에는 이미 **systemd timer**로 자동 운용이 설정되어 있습니다.

- 평일 08:45 자동 시작: `pairbot-start.timer`
- 평일 15:16 자동 중지: `pairbot-stop.timer`

확인:
```bash
systemctl list-timers --all | rg pairbot
```

수동 테스트:
```bash
sudo systemctl start pairbot.service
sudo systemctl stop pairbot.service
```

## 5) 전략 타임테이블(KST)

- **08:50**: 일봉/지표 갱신(ATR20/Noise20/MA5)
- **09:00**: 모순 필터(레버/인버스 중 하나만 MA5 위일 때만 승인) + 목표가 산출
- **09:00~13:00**: 목표가 돌파 시 1회 진입(그룹당 1회)
- **진입 후~15:15**: 샹들리에 트레일링 스탑 감시(ATR*1.5)
- **15:15**: 무조건 전량 청산(Time Exit)

> 참고
> - 지금처럼 장 마감 후(예: 18시)에 실행하면, 15:15 강제 청산 규칙 때문에 빠르게 종료될 수 있습니다.

## 8) 자주 하는 질문

### Q1. 하루에 몇 번 매매하나요?
- 그룹당 최대 1회 진입 → 하루 최대 **2회 매수**
- 청산은 트레일링 스탑 또는 15:15 강제청산으로 최대 **2회 매도**

### Q2. 주문이 안 나가요
아래 순서로 체크하세요.
1) `.env`에서 `KIS_LIVE_ENABLED=1` 인가?
2) `.env`에서 `KIS_LIVE_CONFIRM=YES` 인가?
3) `.env`에서 `KIS_KILL_SWITCH=0` 인가?
4) `STOP_TRADING.flag`가 있는가?

### Q3. 휴장일/주말에도 실행되나요?
- 페어봇은 `utils_holiday.py`로 **휴장일/주말 자동 스킵**합니다.
